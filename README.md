# B2B Lead Extractor: Coffee Shops (Jawa Barat)
Script ini dibuat menggunakan Python untuk menarik otomatis ribuan kontak bisnis kedai kopi secara legal dari OpenStreetMap.

**Teknologi:** Python, Pandas, Requests.

### Hasil Eksekusi (Output)
Script akan memproduksi tabel Excel rapi (Clean Data) yang siap digunakan tim Sales untuk tele-marketing atau WhatsAppblast.
